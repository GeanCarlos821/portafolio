Proyectos

Desarrollar una pagina web en HTML

El objetivo de este proyecto es desarrollar una página web sencilla utilizando únicamente HTML. La página tiene como finalidad presentar información personal, incluyendo una breve biografía, intereses, y formas de contacto. Este proyecto sirve como una introducción práctica al desarrollo web básico..

Desarrollar una pagina web en HTML usan CSS

Crear una página web personal estática utilizando HTML y CSS que funcione como una carta de presentación digital. El sitio mostrará información personal, habilidades, proyectos y medios de contacto, con un diseño visual atractivo y adaptado a distintos dispositivos. Una página web personal visualmente atractiva, profesional y funcional. Este proyecto permite demostrar conocimientos básicos e intermedios de desarrollo front-end, con separación entre estructura (HTML) y presentación (CSS)
